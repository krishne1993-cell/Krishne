# Krishne

A Claude Code repository with Anthropic skills installed for enhanced AI-assisted workflows.

## Skills

Skills are loaded automatically in Claude Code sessions from `.claude/skills/`. Reference any skill by name in your prompt.

### Document Skills
| Skill | Description |
|---|---|
| `xlsx` | Excel spreadsheet creation & editing |
| `docx` | Word document creation & editing |
| `pptx` | PowerPoint presentation creation & editing |
| `pdf` | PDF processing and form handling |
